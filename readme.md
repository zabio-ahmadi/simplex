## **Introduction**

This project is a Python implementation of the simplex algorithm, a method for solving linear programming problems. The algorithm is used to find the optimal solution to a given problem, subject to constraints represented by a system of linear equations or inequalities.

## **Usage**

To use this project, you will need to have Python3 and the numpy library installed on your machine. Once you have these, you can start using the algorithm.

## **Notes**

inside the project there is a folder named **examples** where you cand find predefined problems.
in the main file you have a list of this problems where you cand just passe the problem number to simplex algorithm.
in the example bellow we passed the second problem to simplex algorithm.

```python
problems = [
    'flow_1.txt',                   # 0
    'lp_glaces.txt',                # 1
    'lp_test_2.txt',                # 2
    'lp_test_non_admissible.txt',   # 3
    'lp_test.txt',                  # 4
    'network01_lp.txt',             # 5
    'network02_lp.txt',             # 6
    'network03_lp.txt',             # 7
    'network04_lp.txt',             # 8
    'network05_lp.txt']             # 9

simplex = simplex(problems[1])
simplex.debug(True)
simplex.printSolution()
```

youcan also set the DEBUG flag to true to print the intermediate steps of the algorithm.

```python
simplex.debug(True)
```

The method will print the optimal solution to the problem.

```python
simplex.printSolution()
```
